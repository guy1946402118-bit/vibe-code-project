## 一、项目概述

成长指挥中心是一个集**博客发布**、**每日打卡**、**目标管理**、**AI 技能仓库**、**社交关注**于一体的个人成长飞轮系统。核心理念是将 AI 工具的使用经验沉淀为可复用的"能力宝库"，实现复合工程式的效率增长。

### 技术栈
| 层级 | 技术 |
|------|------|
| 前端 | React 18 + TypeScript + Vite |
| 状态管理 | Zustand |
| 后端 | Express.js |
| 数据库 | SQLite (Prisma ORM) |
| 认证 | JWT + bcrypt |
| 推送 | Service Worker + PWA |

### 启动方式
```bash
# 前端开发
cd D:\GrowthDashboard && npx vite

# 后端服务
cd D:\GrowthDashboard\server && npx tsx src/index.ts

# 构建生产版本
cd D:\GrowthDashboard && npx vite build
```

## 三、核心页面详解

### 🏠 博客首页（成长指挥中心）`/blog`

访客看到：
- 📊 全局统计数据（总用户数、今日打卡、总文章数）
- 🌐 访问态势感知大屏（全球访客实时监测）
- 📈 访问趋势折线图
- 📝 最新博客文章列表
- 🏷️ 热门标签云
- 📈 数据统计（仅博客总数）

登录用户额外看到：
- 🔥 今日焦点卡片（连续打卡天数、番茄钟计时器、活跃目标列表）
- 🛰️ 学习态势感知大屏（六维技能雷达图）
- 🥇 段位进度条 + 快速记录入口
- 📈 个人数据统计（总积分、连续打卡天数）

**键盘快捷键**：`Ctrl+K` 打开智能命令面板

---

### 📝 文章列表 `/blog/posts`

- 所有用户的已发布文章
- 支持按分类筛选（全部/技术/成长/生活/读书/思考）
- 支持标题和内容搜索
- 卡片式响应式布局，显示封面图、阅读量、点赞数
- 支持按最新/最热/最多点赞排序
- 无限滚动分页加载

---

### ✍️ 发布文章 `/blog/new`

- Markdown 编辑器 + 实时预览
- 标题、摘要、分类、标签、封面图
- 草稿自动保存（30秒间隔）
- **发布前审核机制**：点击发布后弹出 7 项自审清单
  - 📌 标题清晰明确
  - 📐 结构逻辑完整
  - 💎 提供具体价值
  - 🎨 排版格式规范
  - ✅ 内容准确可靠
  - 👁️ 考虑读者视角
  - 🏷️ 标签分类恰当
- 显示内容统计（字数/段落/分节标题）和优化建议
- 全部勾选 ✅ 才能发布

---

### 📦 技能仓库 `/skills` 🆕

**灵感来源**：抖音视频「数字游牧人 Samuel」的 AI 第二大脑概念

核心功能：
- 📦 **5 种类型**：提示词、技能、规则、文档、智能体
- 📂 **9 个分类**：前端开发、后端开发、AI/机器学习、效率工具、写作内容、设计创意、数据分析、通用、全部
- ⚡ **快速粘贴**：从任何地方粘贴 AI 提示词文本，自动解析入库
- ⭐ **5 星有效性评分**：筛选真正好用的技能
- 🔄 **使用次数追踪**：识别高频技能和闲置技能
- 🏷️ **标签云检索**：点击标签直接搜索相关技能
- 🔍 **全文搜索**：标题 + 内容索引
- 📊 **排序模式**：最新、最有效、最常用
- 📄 **分页浏览**

**技能仓库统计面板**：
- 技能总数
- 按类型分布（提示词/技能/规则/文档/智能体各多少）
- 累计使用次数

---

### ✅ 每日打卡 `/checkin`

- **5 个打卡分类**：
  - ❤️ 健康（10 积分）
  - 📚 学习（15 积分）
  - 💼 工作（15 积分）
  - 🎯 自律（20 积分）
  - 📝 复盘（25 积分）
- 打卡成功→积分动画+Toast提示
- 连续打卡天数统计
- 📅 月度日历视图（真实打卡数据标记）
- 🔥 热力图组件（真实数据渲染）
- 🎯 今日推进（显示活跃目标进度条）
- 📓 进步本（每日学习笔记）

---

### 🎯 目标管理 `/goals`

- **SMART 目标创建**：
  - 具体目标描述（S）
  - 可量化指标（M）
  - 可达性评估（A）
  - 相关领域（R）
  - 时间绑定（T）+ 里程碑拆分
- 目标分类：习惯、学习、健康、工作、健身
- 优先级：高、中、低
- 进度滑块实时更新
- 目标完成后弹出成就 Toast，引导"写篇复盘文章"
- 📊 四象限矩阵分析
- 进阶目标拆解

---

### 🥇 排行榜 `/leaderboard`

- 时间维度筛选：今日、本周、本月、全部
- 分类筛选：全部、健康、学习、工作、健身、自律
- Top 3 榜单高亮展示（🥇🥈🥉）
- 趋势指示器（↑↓→）
- 🆕 **关注/取关按钮**：
  - 点击「+ 关注」关注好友
  - 点击「✓ 已关注」取消关注
  - 关注后好友动态出现在 ActivityFeed 中

---

### 🏆 成就徽章 `/achievements`

- 徽章收集墙
- 已解锁/未解锁状态
- 成就进度追踪

---

### 🌸 生命之花 `/lifeflower`

- 8 维度人生平衡分析（事业、财务、健康、家庭、社交、学习、休闲、精神）
- 蜘蛛图/雷达图可视化
- 各维度评分 + 改进建议

## 五、数据库模型总览

| Review | 复盘记录 |

| Follow 🆕 | 关注/取关关系 |
| Milestone 🆕 | 里程碑追踪 |
